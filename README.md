##  Project Description
This is a WPF-based Graphical User Interface (GUI) application designed to educate users on cybersecurity. It features a professional terminal-style interface and uses advanced C# concepts to provide an interactive experience.

##  Key Features
* **Delegate System:** Uses a custom `ResponseHandler` delegate to manage bot outputs.
* **Memory & Recall:** The bot remembers your name and the last topic discussed for seamless follow-up questions.
* **Sentiment Detection:** Adjusts its tone if the user expresses worry or frustration.
* **Dynamic Logic:** Uses Generic Lists and the `Random` class to provide varied cybersecurity tips.
* **Terminal Aesthetic:** Designed with a Matrix-style green-on-black color scheme for a cybersecurity feel.

## 🛠️ Built With
* **C# / XAML**
* **WPF (.NET Core)**
* **Visual Studio 2022**

## 📂 Project Structure
* `MainWindow.xaml`: The GUI layout and styling.
* `MainWindow.xaml.cs`: The "Brain" of the bot containing logic and delegates.
* `Assets/`: Folder containing the WAV greeting and ASCII art.
